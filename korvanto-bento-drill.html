Grades

Available Grades

API

KORVANTO BENTO DRILL API

API 13A Grade Bentonite for Oil, Gas & Industrial Drilling

KORVANTO BENTO DRILL API is a high-performance sodium bentonite manufactured to meet API 13A specifications for drilling fluid applications. Processed from carefully selected bentonite deposits, it delivers excellent viscosity, gel strength, suspension characteristics, and fluid loss control required for demanding drilling operations.

When mixed with water, it forms a stable thixotropic drilling mud that helps maintain borehole stability, transports drill cuttings to the surface, cools and lubricates drilling equipment, and minimizes fluid loss into surrounding formations. Its consistent performance makes it suitable for both onshore and offshore drilling environments.

OCMA

KORVANTO BENTO DRILL OCMA

OCMA Grade Bentonite for Drilling Fluid Systems

KORVANTO BENTO DRILL OCMA is a premium drilling-grade bentonite manufactured in accordance with OCMA (Oil Companies Materials Association) specifications. Designed for use in water-based drilling mud systems, it provides reliable rheological properties, good suspension characteristics, and effective borehole support for a wide range of drilling operations.

The product forms a stable drilling fluid that assists in transporting cuttings, sealing permeable formations, reducing fluid loss, and improving drilling efficiency. Its cost-effective performance makes it a preferred choice for water well, geotechnical, and industrial drilling applications.
